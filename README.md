
# Supervised Learning System for Predicting Churn in the Telecommunications Industry

This project is a Data Mining and Machine Learning System developed using:
- Python
- Jupyter Notebook
- Streamlit
- Scikit-learn

## Features
- Data preprocessing
- Exploratory Data Analysis
- Random Forest Model
- Gradient Boosting Model
- Threshold tuning
- Churn prediction system
- Streamlit web interface
- Model comparison

## Dataset
Telco Customer Churn Dataset

## Virtual Environment Setup

### Step 1: Create Virtual Environment
```bash
python -m venv venv
```

### Step 2: Activate Virtual Environment

#### Windows
```bash
venv\Scripts\activate
```

#### Mac/Linux
```bash
source venv/bin/activate
```

### Step 3: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 4: Open Jupyter Notebook
```bash
jupyter notebook
```

### Step 5: Run Streamlit App
```bash
streamlit run streamlit_app/app.py
```

## Research Objective
The project predicts customer churn in the telecommunications industry using supervised learning algorithms to help businesses reduce customer attrition.

## Models Used
1. Random Forest (Human Selected Model)
2. Gradient Boosting (AI Recommended Model)

## Output
- Churn probability
- Churn prediction
- Feature importance
- Business insights
